# To-Do List App

Aplicatie web simpla de gestionare a sarcinilor zilnice, construita cu HTML, CSS si JavaScript vanilla.

## Functionalitati
- Adaugare sarcini cu nivel de prioritate (scazuta / medie / inalta)
- Marcare sarcini ca finalizate
- Stergere sarcini
- Filtrare: toate / active / finalizate
- Salvare automata in localStorage
- Afisarea datei la crearea sarcinii

## Tehnologii
- HTML5
- CSS3 (fara framework)
- JavaScript (ES5 Vanilla)
